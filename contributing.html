2003page Contributing Guide

If you want to contribute to 2003page, please read this guide.

This guide is currently incomplete.

The basics

2003page is hosted on GitHub, thus you will need a GitHub account. It is very easy to sign up on GitHub.

Starting contributing.

After you're finishing signing up, you are able to contribute to 2003page in 2 ways.

Way 1: The slow way (Pull Requests)

The slow way is that you make a pull request every time you make a contribution. This is called the slow way because every commit is delayed for some time due to it being a request.
